## Project: SoliGuard

**Tagline:** *"Catch vulnerabilities before attackers do."*

Solidify is an AI-powered smart contract security auditor built for Web3 developers. It scans Solidity code for vulnerabilities, explains risks in plain English, and generates secure patches.
